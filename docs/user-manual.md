## はじめに

**Lingo Keeper JP** へようこそ！

このアプリケーションは、日本語学習者がストーリーを読みながら楽しく学べる学習プラットフォームです。インタラクティブなストーリー体験とクイズ機能で、あなたの日本語スキルを着実に向上させます。

### 対象レベル
- **JLPT**: N5（初級）〜 N1（上級）
- **CEFR**: A1（入門）〜 C2（熟達）

### 推奨環境
- **ブラウザ**: Chrome、Edge、Safari（最新版）
- **デバイス**: PC、タブレット、スマートフォン
- **インターネット接続**: 必須

---

## 主な機能

### 1. ストーリー体験 📖

6つの日本語ストーリーを読み進め、物語の展開を自分で選択できます。

**特徴**:
- **分岐型ストーリー**: あなたの選択で物語が変わる
- **音声読み上げ**: Google Cloud TTSによる自然な日本語音声
- **ルビ表示**: 漢字にふりがな付き（学習サポート）
- **レベル別**: JLPTレベルに応じたストーリー選択

### 2. クイズで進捗確認 ✅

学習した内容を確認できるクイズ機能。

**特徴**:
- **AI自動生成**: OpenAI GPT-4による高品質な問題
- **即座のフィードバック**: 正解・不正解を瞬時に表示
- **進捗グラフ**: 学習の軌跡を可視化
- **正答率追跡**: レベル別の成績管理

### 3. 学習進捗管理 📊

あなたの学習データをローカルに保存。

**保存データ**:
- 読了したストーリー
- クイズの正答率
- 学習時間
- レベル別の進捗

**注意**: Phase 1ではブラウザのLocalStorageに保存されます（5MB上限）。Phase 2以降、アカウント機能でクラウド同期予定。

## ストーリー体験

### ステップ1: ストーリーを選ぶ

1. トップページで「ストーリー」タブをクリック
2. 6つのストーリーカードが表示されます
3. 各カードには以下の情報が表示されます：
   - **タイトル**: ストーリー名
   - **レベル**: JLPTレベル（N5〜N1）とCEFRレベル
   - **説明**: あらすじ

**選び方のコツ**:
- 初めての方は「N5」レベルから始めましょう
- 自分のレベルより1段階下のストーリーで復習するのも効果的

### ステップ2: ストーリーを読む

1. ストーリーカードをクリックすると、ストーリービューワーが開きます
2. 画面の構成：
   - **上部**: ストーリータイトルとレベル表示
   - **中央**: チャプター本文（ルビ付き）
   - **下部**: 選択肢カード

**読み進め方**:
```
┌─────────────────────────────────────┐
│  新しい朝 (N5 / A1)                 │  ← タイトル・レベル
├─────────────────────────────────────┤
│                                     │
│  今日は月曜日です。                 │  ← チャプター本文
│  田中さんは朝早く起きました。       │  （ルビ付き）
│                                     │
├─────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐        │
│  │ 学校に行く │  │ 公園に行く │        │  ← 選択肢カード
│  └──────────┘  └──────────┘        │
└─────────────────────────────────────┘
```

### ステップ3: 音声で聞く

各チャプターには音声読み上げ機能があります。

1. 本文の横にある **スピーカーアイコン** をクリック
2. Google Cloud TTSによる自然な日本語音声が再生されます
3. リスニング練習に最適！

**音声機能の使い方**:
- **再生**: スピーカーアイコンをクリック
- **停止**: ブラウザの音声コントロールで停止
- **速度調整**: Phase 2で実装予定

### ステップ4: 選択肢を選ぶ

1. チャプター本文を読んだら、下部の **選択肢カード** から1つを選びます
2. 各カードには：
   - **選択肢テキスト**: 行動の内容
   - **説明**: その選択の背景（ホバーで表示）
3. カードをクリックすると、次のチャプターに進みます

**選択肢の例**:
```
┌─────────────────────┐
│  学校に行く         │  ← 選択肢タイトル
│                     │
│  今日は月曜日。      │  ← 説明
│  学校に遅刻しない    │
│  ように急ぎましょう。│
└─────────────────────┘
```

### ステップ5: 物語を完結させる

- 選択を繰り返し、物語を最後まで進めます
- 各ストーリーは複数のエンディングがあります
- 何度でも読み返せます（進捗は自動保存）

**進捗確認**:
- ストーリー一覧で、読了したストーリーには「✓ 完了」マークが表示されます

## クイズで進捗確認

### ステップ1: クイズを開始

1. トップページで「クイズ」タブをクリック
2. **「クイズを開始」** ボタンをクリック
3. ランダムに1問が表示されます

### ステップ2: 問題に回答

1. 画面の構成：
   - **上部**: 問題文
   - **中央**: 4つの選択肢（A、B、C、D）
   - **下部**: 送信ボタン

**回答方法**:
```
┌─────────────────────────────────────┐
│  次の文の（  ）に入る言葉は？       │  ← 問題文
│                                     │
│  彼は毎日（  ）学校に行きます。     │
├─────────────────────────────────────┤
│  ○ A. を                            │  ← 選択肢
│  ○ B. に                            │  （ラジオボタン）
│  ○ C. へ                            │
│  ○ D. で                            │
├─────────────────────────────────────┤
│         [ 回答を送信 ]              │  ← 送信ボタン
└─────────────────────────────────────┘
```

### ステップ3: フィードバック確認

1. 回答を送信すると、即座に結果が表示されます：
   - **正解**: 緑色の「✓ 正解です！」メッセージ
   - **不正解**: 赤色の「✗ 不正解です」メッセージ + 正解表示

**フィードバック例**:
```
┌─────────────────────────────────────┐
│  ✓ 正解です！                        │  ← 正解時
│                                     │
│  「へ」は移動方向を示す助詞です。   │  ← 解説（Phase 2実装予定）
├─────────────────────────────────────┤
│         [ 次の問題 ]                │  ← 次へボタン
└─────────────────────────────────────┘
```

### ステップ4: 進捗グラフを確認

1. クイズページ下部の **「進捗グラフ」** セクションをスクロール
2. グラフには以下が表示されます：
   - **X軸**: 時間（日/週/月）
   - **Y軸**: 正答率（%）
   - **期間切り替え**: 週・月・年ボタン

**グラフの見方**:
- 上向き傾向 → 学習効果が出ています！
- 横ばい → 復習が必要かもしれません
- 下向き傾向 → レベルを1段階下げてみましょう

### クイズの種類

**Phase 1で利用可能な問題タイプ**:
- **読解問題**: ストーリー内容の理解度チェック
- **語彙問題**: 単語の意味・使い方
- **文法問題**: 助詞、活用、文型

**Phase 2で追加予定**:
- **聴解問題**: 音声を聞いて回答
- **発音評価**: スピーキング練習

## 学習のヒント

### 効果的な学習方法

#### 1. ストーリーは繰り返し読む
- 1回目：意味を理解する
- 2回目：音声で聞きながら読む（シャドーイング）
- 3回目：別の選択肢を選んで新しい展開を楽しむ

#### 2. クイズは定期的に受ける
- **推奨頻度**: 1日1回（5〜10問）
- **目標**: 正答率80%以上をキープ
- **復習**: 間違えた問題はメモして復習

#### 3. レベルは段階的に上げる
- 現在のレベルで正答率90%以上 → 次のレベルへ
- 正答率70%未満 → 1段階下のレベルで復習

#### 4. 音声機能を活用
- リスニング力向上に効果的
- 発音練習にも使える（Phase 2で発音評価機能追加予定）

### 学習スケジュール例

**初級（N5-N4）の場合**:
```yaml
月曜日: ストーリー1本（15分）
火曜日: クイズ5問（10分）
水曜日: ストーリー1本（15分）+ 音声復習（10分）
木曜日: クイズ10問（20分）
金曜日: ストーリー復習（20分）
土曜日: クイズ15問（30分）+ 進捗確認
日曜日: 休息 or 自由学習
```

**中級（N3-N2）の場合**:
```yaml
平日: ストーリー1本 + クイズ10問（30分）
週末: ストーリー2本 + クイズ20問（60分）
```

## よくある質問

### Q1. データはどこに保存されますか？

**A**: Phase 1では、ブラウザの **LocalStorage** に保存されます。

**注意事項**:
- ブラウザのキャッシュを削除すると、データも削除されます
- 他のデバイスとの同期はできません
- Phase 2でアカウント機能実装後、クラウド同期に対応予定

**データバックアップ方法**:
現在は手動バックアップ不可。Phase 2以降で対応予定。

---

### Q2. ストーリーは何本ありますか？

**A**: Phase 1では **6本** のストーリーが利用可能です。

**レベル内訳**:
- N5: 2本
- N4: 2本
- N3: 1本
- N2: 1本

Phase 2以降、随時追加予定。

---

### Q3. クイズ問題はどのように生成されますか？

**A**: **OpenAI GPT-4** により自動生成されます。

**品質保証**:
- JLPTレベルに準拠
- 教育専門家による監修（Phase 2予定）
- ユーザーフィードバックに基づく改善

---

### Q4. オフラインでも使えますか？

**A**: いいえ、**インターネット接続が必須** です。

**理由**:
- 音声合成（Google Cloud TTS）
- クイズ自動生成（OpenAI API）
- ストーリーデータ取得

Phase 2以降、オフラインモード（限定機能）を検討中。

---

### Q5. スマートフォンでも使えますか？

**A**: はい、**レスポンシブデザイン** 対応です。

**推奨画面サイズ**:
- スマートフォン: 縦向き推奨
- タブレット: 横向き・縦向き両対応
- PC: 最小幅1280px推奨

---

### Q6. 音声が再生されません

**A**: 以下を確認してください：

1. **ブラウザの音量設定**
   - ミュートになっていませんか？
   - 音量が0%になっていませんか？

2. **ブラウザの権限設定**
   - Chrome: 設定 > プライバシーとセキュリティ > サイトの設定 > 音声
   - Safari: 環境設定 > Webサイト > 自動再生

3. **インターネット接続**
   - Google Cloud TTSへの接続が必要です

4. **対応ブラウザ**
   - Chrome、Edge推奨
   - Safariは一部制限あり

---

### Q7. 進捗グラフが表示されません

**A**: クイズを **最低3回以上** 受けると表示されます。

**表示条件**:
- 最低データポイント: 3回分のクイズ結果
- 期間: 過去7日以内（週表示の場合）

まだクイズを受けていない場合は、「データが不足しています」と表示されます。

---

### Q8. 正答率が低いのですが…

**A**: 以下の方法を試してください：

1. **1段階下のレベルで復習**
   - 現在N3 → N4のストーリー・クイズを受ける

2. **ストーリーを繰り返し読む**
   - 音声機能を活用してリスニング強化

3. **間違えた問題をメモ**
   - 現在は自動保存なし（Phase 2で実装予定）
   - 手動でメモを取ることを推奨

4. **学習頻度を上げる**
   - 1日15分 → 30分に増やす

---

### Q9. アカウント登録は必要ですか？

**A**: Phase 1では **不要** です。

- 登録なしで全機能が使えます
- LocalStorageで進捗保存
- Phase 2以降、アカウント機能実装予定：
  - クラウド同期
  - 複数デバイス対応
  - ランキング機能

---

### Q10. Phase 2ではどんな機能が追加されますか？

**A**: 以下の機能を予定しています：

| 機能 | 説明 | 優先度 |
|------|------|--------|
| ユーザー認証 | アカウント登録・ログイン | P0 |
| データ同期 | クラウドでの進捗管理 | P0 |
| ダークモード | 目に優しい暗いテーマ | P1 |
| 音声認識 | 発音評価（Azure Speech） | P1 |
| ソーシャル共有 | SNSで進捗シェア | P2 |
| アクセシビリティ強化 | WCAG準拠 | P2 |

詳細は `docs/phase2-requirements.md` を参照してください。

## トラブルシューティング

### 問題1: ページが読み込まれない

**症状**: 白い画面のまま何も表示されない

**解決策**:
1. ブラウザをリロード（Ctrl+F5 / Cmd+Shift+R）
2. ブラウザのキャッシュをクリア
3. 別のブラウザで試す（Chrome推奨）
4. インターネット接続を確認

---

### 問題2: 音声が途切れる

**症状**: 音声再生が不安定

**解決策**:
1. インターネット接続速度を確認（推奨: 5Mbps以上）
2. 他のタブを閉じる（メモリ不足の可能性）
3. ブラウザを再起動

---

### 問題3: 選択肢がクリックできない

**症状**: 選択肢カードをクリックしても反応しない

**解決策**:
1. ページをリロード
2. ブラウザのJavaScriptが有効か確認
3. ブラウザを最新版にアップデート

---

### 問題4: 進捗が保存されない

**症状**: ストーリーやクイズの進捗が消える

**原因**:
- ブラウザのプライベートモード使用中
- LocalStorageが無効
- ブラウザのキャッシュ削除

**解決策**:
1. 通常モードでブラウザを使用
2. ブラウザ設定でLocalStorageを有効化
3. Phase 2のクラウド同期機能を待つ

---

### 問題5: モバイルでレイアウトが崩れる

**症状**: スマホ画面でテキストが見切れる

**解決策**:
1. ブラウザを最新版にアップデート
2. 画面を縦向きにする
3. ブラウザの拡大率を100%にリセット
